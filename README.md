# AI-ML-Internship-Task-5
# ❤️ Heart Disease Prediction using Decision Trees & Random Forests

## 📌 Project Overview

This project applies **Decision Tree** and **Random Forest** classifiers to predict heart disease using a real-world Heart Disease dataset.

The objective is to understand tree-based models, overfitting control, ensemble learning, feature importance, and cross-validation.

This project is completed as part of **AI & ML Internship – Task 5**.

---

## 🎯 Objective

- Train Decision Tree classifier
- Visualize tree
- Control overfitting using max depth
- Train Random Forest
- Compare accuracy
- Analyze feature importance
- Perform cross-validation

---

## 🛠 Tools Used

- Python  
- Pandas  
- Matplotlib  
- Scikit-learn  
- Kaggle Notebook  

---

## 📂 Dataset

Heart Disease Dataset (heart.csv)

Target Column:

- `0` → No Heart Disease  
- `1` → Heart Disease  

---

## ⚙️ Steps Performed

### 1. Load Dataset

Loaded CSV from Kaggle input directory.

---

### 2. Train-Test Split

80% Training  
20% Testing  

---

### 3. Decision Tree

- Trained Decision Tree Classifier
- Evaluated accuracy
- Limited tree depth to reduce overfitting
- Visualized decision tree

---

### 4. Random Forest

- Trained Random Forest Classifier
- Compared accuracy with Decision Tree

---

### 5. Feature Importance

Extracted feature importance from Random Forest to identify most impactful attributes.

---

### 6. Cross Validation

Applied 5-fold cross validation for better performance estimation.

---

## 📊 Results

Outputs include:

- Decision Tree Accuracy  
- Limited Depth Tree Accuracy  
- Random Forest Accuracy  
- Feature Importance  
- Cross Validation Score  

Random Forest achieved higher accuracy due to ensemble learning.
